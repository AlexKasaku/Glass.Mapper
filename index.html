Glass.Mapper

Glass.Mapper is an open source object mapping framework for the most popular .NET CMS systems on the market.

Get Started »

Glass.Mapper has been the redevelopment of the very popular Glass.Sitecore.Mapper project. This project aims to create a more robust and more flexible solution that not only works with many different CMS's. Currently supported are:

  • Sitecore
  • Umbraco

A key feature of the implementation is the introduction of pipelines for most of the tasks carried out by Glass.Mapper allowing you to swap in your preferred solution, for example if you don't like the standard way to create a concrete object then you can implement you own and swap it in.
