Stable

Stable releases of Glass.Mapper are available from Nuget.org.

Pre-Releases

You can download the pre-releases of Glass.Mapper from our own build server by setting up a custom package source for Nuget in Visual Studio:

http://build.glass.oss.eduservlab.net/guestAuth/app/nuget/v1/FeedService.svc/

Pre-release versions are build every night by our nightly build assuming we have changed some code in the source.

Our build server has been kindly hosted by Eduserv.

How to setup a package source

The following instructions will show you how you can setup a package source for Nuget in Visual Studio.

  1. Open Visual Studio
  2. Click on the Tools menu
  3. Click the Library Package Manager sub-menu
  4. Click the Package Manager Settings Options.
  5. You should now see the Options window and it should be open at Package Managers/General
  6. Select the next option down Package Sources. You should see on the right hand side under Available Package Sources the Nuget Official package source.
  7. Click the plus symbol in the top right of the screen, this should add a package source.
  8. Select the new source and in the Name box at the bottom of the window enter Glass Build Server.
  9. Copy the URL above into the Source box.
  10. Click the Update button.
  11. Click Ok to close the Options window.

Now that you have setup Nuget to use the Glass Build Server as a second package source you should see it appear in the Nuget Package Manager. Within your project open the Package Manager screen and you should see the Glass Build Server listed under Online:
