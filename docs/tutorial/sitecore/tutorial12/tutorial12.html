Tutorial 12 - Sitecore MVC

It is possible to get Glass.Mapper to generate the models used by Sitecore MVC.

Models used by Sitecore MVC are generated by the mvc.getModel pipeline. When you installed Glass.Mapper.Sc a config file called Glass.Mapper.Sc.Mvc.config was inserted into the /App_Config/Include directory, this config file creates an additional processor to the mvc.getModel pipeline to create models from Glass.
