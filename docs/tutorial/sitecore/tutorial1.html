Tutorial 1 - Setup

This tutorial will take you through how to setup Glass.Mapper to work with Sitecore.

The first task when using Glass is to configure your model classes so they can be used by Glass.Mapper. You have two options for configuring your class class, either fluent or attributed, you can read more about these in the Concepts section. For this demo we will be using fleunt configuration.
