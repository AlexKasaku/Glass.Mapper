Contribute

You can contribute to the Glass.Mapper documentation to make it even better. Follow four simple steps:

  1. Fork the Glass.Mapper project on GitHub
  2. Switch the gh-pages branch
  3. Update the documentation
  4. Send us a pull request

Read Using Jekyll with Pages on GitHub for more information on how to edit the documentation.
