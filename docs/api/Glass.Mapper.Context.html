{{page.title}}

The Context class controls the configuration and the data loaded by Glass.Mapper. You can have a single Context or multiple Contexts with different configuration.

Constructors

Name Description

Properties

Name Description
ResolverFactory

Static

The factory class that is used to create the dependency resolver. By default the Context class will use the CastleDependencyResolverFactory class as the default factory.

If you wish to use a different IOC container set your custom factory here.

Default

Static

The default Context used by service created by Glass.Mapper.

Contexts

Static

Collection of all Contexts that have been created by Glass.Mapper.

Methods

Name Description
Create(
IGlassConfiguration glassConfig,
string contextName,
string isDefault = false)

Static

Creates a new Context and adds it to the Contexts list. If the Context is default it will also be assigned to the Default property.

Clear() Removes all Contexts in the Contexts properties and sets the Default Context to null.
