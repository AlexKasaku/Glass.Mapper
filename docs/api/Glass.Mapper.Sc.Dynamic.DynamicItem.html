The DynamicItem is used to supply support for the dynamic c# keyword.

The DynamicItem is used to supply support for the dynamic c# keyword. It allows you to map basic Sitecore fields and item meta-data. This means that you don't need to create a model to pull data from Sitecore. You can only read data using the dynamics keyword, it will not save any data.

Fields

Field support is supplied by using the property name as the field name on the specified item. A property will only be used as a field name if it doesn't match any of the reserved property names detailed further down.

When a field is mapped by a dynamic item it is automatically sent via the Sitecore FieldRender control and therefore will support Rich Text, Image and Link fields. It will also support the page editor. The example below shows how to map a field called Title and the Updated date field: 

        var context = new SitecoreContext();
        var item = context.GetCurrentDynamicItem();
        
        string title = item.Title;
        string updated = item.__Updated;

Metadata

Metadata support is supplied by support the enum values defined on the Glass.Mapper.Sc.Configuration.SitecoreInfoType type. For example to map the items display name we would specify a property called DisplayName: 

        var context = new SitecoreContext();
        var item = context.GetCurrentDynamicItem();
        
        string displayName = item.DisplayName;
        int version = item.Version;

Relationships

Simple relationships can be mapped by using the Parent and Children properties. These properties return another dynamic item and an IEnumerable<dynamic> respectively. 

        var context = new SitecoreContext();
        var item = context.GetCurrentDynamicItem();
        
        var parent = item.Parent;
        string parentPath =  parent.FullPath;
        
        foreach(var child in item.Children){
         
        }

Item Id

The item ID can be returned using the Id property. This is returned as the ID type. 

        var context = new SitecoreContext();
        var item = context.GetCurrentDynamicItem();
        
        var id = item.Id;

Dynamic item collections

Dynamic item support has support for simple Linq statement support. The best place to see what Linq functions can be achieved with dynamic items is to check out the integration tests . You will notice that we have to call the methods Dy.Fc and Dy.FcT methods to use Linq support. This is due to the way that C# implements dynamics. 

        var context = new SitecoreContext();
        var item = context.GetCurrentDynamicItem();
        
        var childByName = item.Children.First(Dy.Fc(x => x.Name == "Child2"));
        var secondChild = item.Children.ElementAt(1);
