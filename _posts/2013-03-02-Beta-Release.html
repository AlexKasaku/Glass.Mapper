We will soon be ready to release the first beta of Glass.Mapper version 3 and a lot has changed since the original Glass.Sitecore.Mapper.

Below are just a few of the improvements and features in version 3:

  • Designed to work with multiple different CMS systems, write your business logic once and use it everywhere.
  • A build server that will provide nightly builds of the latest releases
  • Using a proven DI container (Castle Windsor)
  • Introduced pipelines so that you can customise how objects are built
  • Trello board to track what we are doing
  • Everything including the main site is now open so everyone can contribute
  • More and better unit tests
  • A better code structure

Please contact us if you have any questions about Glass.Mapper or want to get involved.
