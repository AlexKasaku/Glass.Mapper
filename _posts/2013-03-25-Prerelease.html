Pre-Release Versions - 25 March 2013

With the launch of Glass V3 we have introduced a build server, which has been kindly hosted for use by Eduserv. This will allow you to download the latest Glass versions before they are made available on Nuget.org.

This server is up and running now so if you want to get Glass V3 before it is available on Nuget follow the instructions on our Download page under the Pre-Release section.

In the future we hope this will be a great way for everyone to get bug fixes and feature improvements before the are officially pushed to Nuget.org as stable.

This Pre-release contains just the Sitecore components but we hope to bring the Umbraco components soon.
